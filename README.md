Sorting Service

## Run
```bash
# to install dependencies
yarn 

# for dev mode with live reload
yarn dev 

# or this to produciton mode
yarn start
```

Open [http://localhost:3000](http://localhost:3000)

## Tests

1. `yarn`
2. `yarn test`
